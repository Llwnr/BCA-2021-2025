// NOTE: <HTML> = tags, if given <HTML> then HTML is the element, if <HTML /> it should have a opening tag like <HTML> and closing tag like </HTML>


A.  <html />    =   Used to indicate that the nested texts are Hyper Text Markup Language

B.  <head />    =   Used to indicate that the nested texts belong to the header of the HTML

C.  <body />    =   Used to indicate the overall body of the BROWESER Web Page

D.  <title />   =   Used to give name on the TAB LIST of BROWSER

E.  <p />       =   The nested texts works as text in paragraph

F.  <br>        =   This tag is used to break one line/ENTER

G.  <hr>        =   Used to make a straight horizontal line

H.  <strong />  =   Used to make the nested TEXT BOLD

I.  <center />  =   Used to place nested text in center alligned way

J.  <hN />      =   Used to indicate imporatant text which is largest among N = 1 to 6
